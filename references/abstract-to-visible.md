# 抽象信息转可见画面

把心理、评价、关系结论和抽象氛围改写为摄像机或麦克风能够记录的证据。默认保持剧情事实、人物动机和事件结果不变；不要用“更有戏”作为擅自加戏的理由。

## 适用范围

在以下情况加载本手册：

- 原文直接说明人物内心。
- 原文用作者口吻解释表情或动作的“真正含义”。
- 原文只给出情绪、气氛、关系或权力结论。
- 已有分镜写了“压迫感”“宿命感”“高级感”等不可执行词。
- 镜头的叙事目的只存在于导演解释栏，画面本身无法传达。

不要在本阶段决定景别、机位或切镜。先生成可见事件，再交给调度和原子分镜阶段处理。

## 核心判定

对每句话执行“关掉解释栏测试”：

> 删除心理词、情绪标签和作者解释后，观众还能否仅凭画面与声音判断发生了什么？

若不能，补充证据；不要补充新的剧情事实。

使用以下四级判定：

| 等级 | 状态 | 处理 |
|---|---|---|
| A | 已经是可见、可听事件 | 保留并压缩歧义 |
| B | 有情绪词，也有明确表演动作 | 保留动作；把情绪词降为表演参考 |
| C | 只有心理、判断或氛围结论 | 转成可见证据 |
| D | 转译需要新增关键事实 | 停止写入正式分镜，标记需授权 |

情绪词本身不是错误。错误是让情绪词替代可执行表演。

## 转译流程

按以下顺序处理每个抽象句：

1. 锁定原文来源、事实 ID 和修改权限。
2. 提取不可改变的事实：谁、知道什么、想做什么、发生了什么结果。
3. 区分“必须被观众理解”与“只供演员参考”。
4. 找到人物可以影响的对象：身体、道具、距离、空间、声音或环境。
5. 选择最少的一组可见证据。
6. 写明开始状态、动作和结束状态。
7. 检查动作是否在现实或模型中可执行。
8. 检查是否偷加了身份、关系、证据、危险或事件结果。
9. 删除解释性尾句，再做一次“关掉解释栏测试”。
10. 将仍有歧义的内容标记为人工确认，不要用更多文学词掩盖。

推荐句法：

```text
主体 + 从开始状态执行一个明确动作 + 作用于对象或空间 + 形成可见结束状态
```

例如：

```text
她的工牌原本挂在胸前。
她摘下工牌，压在辞职信上，把两样东西推到经理面前。
镜头结束时，工牌和辞职信停在经理手边。
```

## 六种可见证据

按优先级选择。优先使用能推动事件的动作，不要堆叠所有渠道。

### 1. 有结果的身体动作

让动作改变人物、物体或空间状态。

```text
弱：他愤怒地站起来。
强：他推开椅子站起，椅脚撞上身后的柜门。
```

### 2. 道具状态变化

让抽象决定留下可追踪的物理结果。

```text
弱：她决定断绝关系。
强：她从钥匙圈上取下公寓钥匙，放到餐桌中央。
```

### 3. 人物距离与空间关系变化

用靠近、退开、阻挡、让路、占位或离场表达关系变化。

```text
弱：他开始防备她。
强：她靠近时，他把文件夹合上，并用身体挡住桌上的手机。
```

### 4. 可执行的面部与手部变化

写肌肉、视线、呼吸或手部动作，不写不可见的含义。

```text
弱：她露出意味深长的笑。
强：她看一眼录音笔，嘴角抬起；对方伸手时，她把录音笔收回掌心。
```

不要依赖“眼神复杂”“表情微妙”完成关键剧情信息。

### 5. 可听信息

使用停顿、吞咽、呼吸、敲击、门锁、提示音或一句必要对白。

```text
弱：他意识到银行账户被冻结。
强：手机连续弹出三条“交易失败”提示音；他再次按下付款键，屏幕仍显示失败。
```

### 6. 环境的物理反应

只使用人物行为能够引发或环境真实发生的变化。

```text
弱：空气凝固了。
强：会议室里没人接话，只剩投影仪风扇声；秘书停下记录。
```

“环境反应”不是让天气、灯光或物品无因变化来替人物表达情绪。

## 常见抽象类型

### 心理结论

原文直接告诉观众人物想到、明白、后悔或决定了什么。

处理方式：

- 若结论会引发行动，拍行动。
- 若结论来自证据，拍人物发现证据的过程。
- 若结论暂时不应被观众知道，保留为演员备注，不强行外化。

### 作者解释

典型句式：

```text
这个冷笑不是得意，而是……
她的沉默代表……
他看似退让，其实……
```

删除解释，改写为能够让观众得出同一结论的行为链。

### 关系与权力结论

把“谁压制谁”“谁失去发言权”转成座位、文件、门口、发言顺序、物品控制或他人反应。

### 抽象氛围

把“压迫、疏离、温暖、危险”转成具体光线、距离、遮挡、声音、节奏和空间条件。

不要只替换同义词。把“压迫感”改成“阴影压迫感”仍然不可执行。

### 时间与记忆

把“她想起过去”转成触发物、声音、影像、明确闪回或人物停顿后的行为。是否插入闪回属于剪辑决策，不在本阶段擅自决定。

### 象征和主题

允许象征，但象征物必须真实存在并服务当前行动。

```text
坏：破碎的镜子象征她破碎的人生。
好：她贴好镜面裂缝，镜中仍把她的脸分成两半。
```

不要在画面描述中解释象征意义。

## 动作设计规则

### 让动作有起点和终点

```text
坏：她玩着戒指，心中挣扎。
好：她转动戒指两圈，停下，把戒指摘到指节位置；门外脚步声响起，她又将戒指推回原位。
```

### 让关键动作改变状态

把“看、想、感到”升级为能够改变信息或关系的动作。

```text
弱：他看着合同，感到震惊。
强：他翻到合同末页，手指停在自己的伪造签名上；下一页没有继续翻。
```

### 让动作符合人物权限

人物不能无依据获得信息或控制物品。

```text
坏：她意识到保险柜里没有钱，于是冷笑。
好：她输入密码，拉开保险柜；里面只剩一个空文件袋。
```

### 控制动作数量

为一个抽象结论选择一到两个最有辨识度的动作。不要把所有肢体反应塞进同一镜头。

```text
过载：她握拳、咬唇、红眼、后退、撞椅子、抬头冷笑，再把合同撕碎。
精简：她将合同沿签名处撕成两半，放回对方面前。
```

### 使用可重复表演

动作要能在多个 Take 或 Attempt 中重复，避免只写“恰到好处地”“微妙地”“带着三分……”。

### 避免陈词滥调自动映射

不要把每种情绪固定映射为同一个动作：

- 紧张不一定握拳。
- 悲伤不一定流泪。
- 愤怒不一定摔东西。
- 心虚不一定移开视线。
- 反击不一定冷笑。

根据人物、道具、空间和前后行动选择证据。

## 对白与声音的使用边界

优先用动作表达能被动作表达的内容。仅在以下情况使用对白：

- 信息无法从当前画面合理推断。
- 人物必须公开立场并产生对手反应。
- 台词本身是证据、威胁、承诺或规则。
- 沉默会改变原剧情事实。

避免把内心说明改成直白自述：

```text
坏：
“我终于不再忍耐了，我要反抗你。”

好：
她把工牌压在辞职信上：“从今天起，你没资格替我签字。”
```

不要使用旁白替代缺失的画面设计：

```text
坏：
旁白：她不知道危险正在靠近。

好：
她背对门整理药盒；门把手缓慢向下转动。
```

若关键信息只能依赖声音理解，标记 `required_for_comprehension`，并要求字幕或可访问替代，不要让信息悄悄消失。

## 高频好坏例

### 决定与反抗

```text
坏：她终于不再隐忍。
好：她关掉录音笔，把工牌和辞职信一起推到经理面前。
```

```text
坏：他决定与父亲决裂。
好：他把家族印章放回父亲桌上，提起自己的行李箱走向门口。
```

```text
坏：她用沉默完成了反抗。
好：主持人第三次把话筒递来，她没有接，转身将证据投到大屏幕。
```

### 发现与怀疑

```text
坏：他意识到自己被出卖了。
好：他在合同末页看到合伙人的签名，抬头看向门边；合伙人避开他的视线。
```

```text
坏：她开始怀疑丈夫。
好：她对照丈夫手机里的酒店收据与墙上日历，用笔圈住同一个日期。
```

```text
坏：他知道门外的人不是母亲。
好：门外的人敲出三短一长；他母亲一直使用两短两长。他的手停在门锁上。
```

### 权力变化

```text
坏：会议室的权力关系逆转了。
好：董事长把表决文件转向她，原本发言的经理放下遥控器，退到屏幕旁。
```

```text
坏：所有人终于开始尊重她。
好：她走向主位时，两名董事同时收起桌上的文件，为她让出座位。
```

```text
坏：他被彻底压制。
好：他伸手拿合同，律师先一步按住合同，将签字笔推给对面的女人。
```

### 情绪和关系

```text
坏：她很失望。
好：她把准备好的两只酒杯中的一只收回柜子，只留下自己面前的一只。
```

```text
坏：他心虚地笑了。
好：她问到转账记录时，他的笑停住，拇指按灭手机屏幕。
```

```text
坏：两个人之间产生了隔阂。
好：电梯门打开，她先走进去；他迈出一步又停下。电梯门在两人之间合拢。
```

```text
坏：她终于释然。
好：她删掉未发送的语音，取下抽屉里的旧合照，放进碎纸机。
```

### 氛围

```text
坏：房间里充满压迫感。
好：低矮顶灯只照亮桌面，三个人站在她身后；唯一出口被最靠门的人挡住。
```

```text
坏：空气中弥漫着宿命感。
好：墙上的婚礼倒计时归零，楼下礼炮同时响起；她仍穿着沾血的白衬衫。
```

```text
坏：现场异常安静。
好：主持人停下讲话，观众席无人鼓掌，只剩麦克风回授的低鸣。
```

### 复杂情绪

```text
坏：她又爱又恨地看着他。
好：她替他扣好松开的袖扣，随后把他的手从自己肩上移开。
```

```text
坏：他因胜利而感到空虚。
好：领奖台灯光亮起，他举起奖杯；台下为家属预留的座位仍然空着。
```

```text
坏：她想哭却忍住了。
好：她吸气，张口准备说话，声音没有出来；她低头把皱起的纸页抚平后再抬头。
```

### 危险与悬念

```text
坏：她不知道危险正在逼近。
好：她背对门清点药片；门缝下出现一双停住的鞋。
```

```text
坏：他感到死亡的气息。
好：电梯楼层数字停在负二层，门打开后，走廊灯逐盏熄灭。
```

```text
坏：事情远没有结束。
好：警方封存证物后，桌下第二部手机亮起同一段监控画面。
```

## 特殊情况

### 不应外化的秘密

若故事要求观众此刻不知道人物真实想法，不要设计泄密动作。只给演员可执行但不定向的行为：

```text
可用：他听完消息后折好餐巾，放到盘子左侧。
不用：他露出计划得逞的笑。
```

### 推理和复杂信息

不要用一个“恍然大悟”的表情代替推理链。展示最少的证据连接：

```text
照片上的反光
→ 当前房间相同的窗格
→ 人物把照片转向窗户比对
```

若需要多个独立信息镜头，把它们交给 Coverage 和原子分镜阶段拆分。

### 梦境、幻觉和主观体验

可以拍主观画面，但必须标记其叙事状态，避免被误当作客观事实：

```text
subjective_state: dream | memory | hallucination | imagined
```

### 无对白表演

让动作承担信息，不要用括号补写心理：

```text
坏：她把饭盒放下（意识到母亲不会回来）。
好：她摆好两副筷子；等到墙钟越过十二点，她收走其中一副。
```

### 喜剧

优先使用可见误差、重复、错误目标和动作结果。不要写“气氛突然滑稽”。

```text
坏：他尴尬得想找地缝。
好：他举着花束敲错门；门内走出一整队参加葬礼的人。
```

## 硬失败

出现以下任一项，退回视觉转译阶段修复：

- 关键叙事信息只存在于心理词、作者解释或导演备注。
- 使用“不是……而是……”解释画面真实含义，画面本身没有证据。
- 动作无法被摄像机或麦克风记录。
- 转译凭空增加身份、证据、关系、危险、犯罪、死亡或事件结果。
- 用一句直白自述替代本可通过动作表达的核心转折，并改变人物说话方式。
- 使用无因的天气、灯光、物品变化替人物表达情绪。
- 用“眼神复杂”“意味深长”“高级感”等模糊词承担关键事实。
- 只有动作起点，没有可判断的结果；或只有结果，没有动作来源。
- 为表达一个心理结论塞入多个不连续动作，导致镜头不可执行。
- 梦境、回忆、幻觉或想象未标记主观状态，可能被当成客观事实。

## 软警告

以下情况先标记警告，再结合上下文判断：

- 使用情绪词，但已有明确动作支持。
- 使用眼神或微表情传递非关键细节。
- 使用常见动作陈词滥调，但符合人物和场景。
- 同一抽象句使用两个以上证据渠道，可能过度解释。
- 为了画面化增加了原文未写明的普通生活动作。
- 关键含义依赖极细小道具，竖屏或低分辨率可能看不清。
- 使用沉默表达信息，但没有其他人物或环境反应。
- 使用象征物，但它未在 Story Bible 或场景中建立。
- 对白与动作表达完全重复，可能显得啰嗦。
- 画面可以拍，但动作难以稳定重复或生成。

## 输出检查表

提交给调度和原子分镜阶段前逐项确认：

- [ ] 保留了原文锁定事实。
- [ ] 所有新增内容都属于表现补全，或带有授权 ID。
- [ ] 关键心理信息已转换为可见或可听证据。
- [ ] 每个动作都有明确主体、对象和物理结果。
- [ ] 已写清开始状态和结束状态。
- [ ] 删除解释栏后，观众仍能读懂主要事件。
- [ ] 没有用抽象风格词代替灯光、空间或声音条件。
- [ ] 没有为了表达情绪堆叠过多动作。
- [ ] 秘密、误导和主观画面没有提前泄露剧情。
- [ ] 复杂证据链已标记供 Coverage 和原子分镜阶段拆分。
- [ ] 关键声音信息已标记为理解所必需。
- [ ] 所有无法在当前权限内解决的问题都已退回人工确认。
